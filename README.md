## Paikallisesti ajettavissa Denolla:
Portissa 8080
```
deno run --allow-read --allow-net --allow-env src/server.ts
```
Jossain muussa portissa

```
deno run --allow-read --allow-net --allow-env src/server.ts --port=3000
```
